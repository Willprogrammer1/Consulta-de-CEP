# Consulta-de-CEP
O projeto consiste em uma aplicação web para consulta de CEP, desenvolvida com HTML, CSS e JavaScript. Utiliza a API ViaCEP para obter dados de endereço e exibe a localização em um mapa interativo. Ferramentas de Inteligência Artificial foram usadas como apoio ao aprendizado e revisão de código, com total compreensão das soluções aplicadas.
